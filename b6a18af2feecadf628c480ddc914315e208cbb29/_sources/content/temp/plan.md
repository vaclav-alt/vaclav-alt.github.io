# Průběh

## 1. týden

- Úvod do jazyka, zařazení do kontextu ostatních programovacích jazyků; historie; sféra použití, PEP.
- Interpretace vs. kompilace;
- vývojová prostředí,
- instalace interpreta a virtuálních prostředí, první program,
- orientační seznámení se syntaxí.


## 2. týden

Python data model, mutability a immutability; seznámení se základními datovými typy a jejich dělením: číselné datové typy, integer vs. floating point numbers a jejich specifika v Pythonu; logický typ; kolekce a sekvence, list, tuple, dictionary; formátovaný výstup.


## 3. týden

**splněné lekce:** Přehled základních datových typů; Tuple, list, dictionary; String

Python memory management, garbage collection. Podmínky a logické výrazy; cykly a iterace; funkce, složitější signatura, anonymní funkce.

## 4. týden

**splněné lekce:** Podmínky a logické výrazy; Cykly a iterace

Typing: Python jako dynamicky typovaný jazyk, type hints, docstring, vlastní datové typy; konstrukce match vs. známý switch; výjimky a mechanismus jejich zpracování.

## 5. týden

**splněné lekce:** Základní struktura funkcí; Složitější funkce

Načítání ze souboru, context manager a with statement, načítání ze souborů běžných formátů. Užitečné a běžně užívané balíčky: os, sys, pathlib, datetime, logging.

## 6. týden

Dekorátor jako univerzální wrapper, jeho význam v Pythonu a syntactic sugar. Úvod do OOP: zavedení základních pojmů, konstruktor vs. inicializátor, class vs. instance attributes, instance vs. class vs. static methods; magic methods, iterátor, generátor, context manger; zapouzdření, omezení přístupu, getter a setter; dataclass.

## 7. týden

**splněné lekce:** OOP - Základní pojmy; Zapouzdření; Výjimky

Pokračování OOP: dědičnost, vícenásobná dědičnost, diamond problem a Method resolution order; abstraktní třídy, virtuální metody, abstraktní metody; duck typing a protocol class; mixin; úskalí vícenásobné dědičnosti.

## 8. týden


Import a jeho funkce; context a scope; closure. Modul vs. balíček, tvorba vlastního modulu/balíčku, instalace, co znamená instalace. Dokumentace a její automatické generování.

## 9. týden

Numpy: sféra zaměření, vektorizované operace a jejich výhoda oproti cyklu v Pythonu; co znamená, že je Python pomalý. Matplotlib: grafická knihovna, konstrukce grafů základních typů; fraktály; (animace)

## 10. týden

Balík requests, práce s věřejnými API a stahování dat. práce s DataFrames, základní statistika, filtrování a čištění dat v balíku pandas

## 11. týden

User interface v Pythonu, přehled knihoven pro CLI (argparse, cmd) a GUI (curses, Tkinter, PyQT), tvorba UI pro programy z předchozích týdnů.


## 12. týden

Testování v Pythonu, mocking, refactoring; debugging.
