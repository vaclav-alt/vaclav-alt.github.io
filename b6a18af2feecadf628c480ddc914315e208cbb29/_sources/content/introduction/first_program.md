# První program

V libovolném textovém editoru si připravíme textový soubor `hello.py`, do kterého umístíme
```python
print("Hello World!")
```
a uložíme do naší pracovní složky. V konzoli pak program spustíme tak, že interpretu dáme cestu k souboru

```console
# windows
> python hello.py
Hello World!

# linux/mac
$ python3 hello.py
Hellow World!
```
